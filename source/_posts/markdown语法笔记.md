# markdown语法笔记

## 插入

### 链接

1. 行内链接：```[显示文本](链接地址 "可选标题")```
2. 引用式链接：
   ```markdown
   [显示文本][链接标识]

   [链接标识]: 链接地址 "可选标题"
   ```
3. 直接显示URL：```<https://markdown.org>```
4. 图片链接：```[![图片替代文本](图片URL)](链接地址)```

### emoji

* ```win + .```打开系统自带的emoji搜索
* [emoji中文网](https://www.emojiall.com/zh-hans)等复制粘贴
* 使用标签符号简码，需要安装Markdown Emoji插件，语法```:joy:```，效果:joy:。（此方法hexo不支持）
